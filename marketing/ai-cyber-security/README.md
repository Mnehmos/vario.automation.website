## Content Strategy

### Honeypot Educational Material

Educational content that demonstrates attack vectors to establish credibility and drive consulting inquiries:

1. **Attack Pattern Library** (`/honeypots/`)
   - Sleeper injections (delayed payload attacks)
   - Persistence attacks (self-propagating payloads)
   - Exfiltration triggers
   - Social engineering triggers
   - Escalation patterns

2. **Detection Guides** (`/detection/`)
   - Behavioral anomaly detection
   - Tool call auditing
   - Conditional pattern scanning
   - Honeypot tripwires

3. **Defense Frameworks** (`/defense/`)
   - Ingestion quarantine patterns
   - Tool manifests and risk profiles
   - Execution provenance
   - Sandboxed reader architectures
